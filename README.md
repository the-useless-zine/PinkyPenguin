## 一些寫給主編看的，懂？

### 發刊時間
  每雙週一台灣時間早上 8 點

### 標題引文
  當期主編自己決定標題與引文，可以不要有引文。

### 內容
  - 基本上為十篇上下的連結與其介紹，另外加上每次開頭與讀者聊天的前文部分。
  - 連結上要寫為什麼寫這篇文，你只要寫得出來要分享什麼都可以。

### 編輯方法
  - 我們是用 curated 搜集內容，可以直接進入 curated 後台將連結跟內文丟進去編輯即可。
  - 沒用小誌有 slack，其中的 #zine 頻道可以在未進入後台的情況下儲存連結，有需要可以加入。
